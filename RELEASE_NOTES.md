# 🚀 Ask CPT - v1.0.1 Release Notes

**The Ultimate AI Coding Assistant is Now Multi-Platform!**

We are thrilled to announce that **Ask CPT** has evolved. Originally built as the ultimate LeetCode companion, Ask CPT has been completely redesigned from the ground up to support **ALL major auto-coding platforms and technical assessment environments**.

### 🌟 What's New?

#### 🌐 Universal Platform Support
Ask CPT's DOM parsing engine now supports intelligent "Blind Reading." It can universally extract problems, examples, and active coding languages from any website. You can now use Ask CPT natively on:
- **LeetCode** (Problems, Assessment, Explore)
- **HackerRank** (Assessments & Practice)
- **Codeforces**
- **GeeksforGeeks**
- **Coderbyte & CodeSignal**

#### 🛡️ Advanced Invincibility Engine
We've upgraded our core injection framework to survive intense Single Page Application (SPA) routing, full-screen lock modes, and isolated multi-pane assessment environments without ever failing.

#### 🤖 Universal Error Scanner
Our smart error scanner doesn't just read LeetCode logs anymore. It dynamically catches exceptions (`SyntaxError`, `Wrong Answer`, `Timeout`) rendered in almost any standard web console or UI toaster—instantly mapping failures to your active codebase.

#### ⚡ Groq Llama-3.3 70B Integration
Lighting fast generation! By leveraging the incredible speed of Groq's engine, code generation and auto-fixes happen in **sub-second** times, keeping you in the ultimate flow state.

---

### 🛠️ Updating to v1.0.1
1. Open Chrome and go to `chrome://extensions/`
2. Ensure **Developer Mode** is toggled ON.
3. Click **Update** or click the **Reload 🔄** icon on the Ask CPT card.

Enjoy the new universal AI coding experience! 🎯
